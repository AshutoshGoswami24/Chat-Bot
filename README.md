### CONFIGS VARIABLES

* `BOT_TOKEN`  - Get bot token from @BotFather
* `API_ID` - From my.telegram.org
* `API_HASH` - From my.telegram.org
* `ADMIN` - Admin user id
* `PICS` - Start message photo. `Optional`
* `START_TXT` - Start Txt
* `ALL_TXT` - Your Channal Info
* `I-Am-Ashu` - `True` Don't Try To Change Its Required
